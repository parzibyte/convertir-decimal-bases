Para convertir un número decimal a cualquier base, hay que dividir el decimal entre esa base y sacar el residuo junto con el cociente. En cada paso, el residuo es el dígito que compondrá al resultado, y el cociente es el nuevo número decimal que seguiremos dividiendo hasta que el mismo sea 0
